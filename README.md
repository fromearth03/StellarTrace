# StellarTrace
StellarTrace is a custom-built search engine designed to efficiently index and retrieve large-scale data. Its purpose is to provide fast and relevant search results across diverse content.

Include folder contains hpp files where main logic is defined and use main.cpp to run code i have given example code there
